About Me

안녕하세요. 한국외국어대학교 컴퓨터공학부 4학년에 재학 중인 Geonwoo입니다. 조기 졸업을 예정하고 있으며 컴퓨터구조 및 시스템 분야 대학원 진학을 목표로 학업을 이어가고 있습니다.

학부 과정에서 운영체제, 컴퓨터구조, 병렬처리와 같은 low-level 시스템 과목에 큰 흥미를 느끼게 되었고, 소프트웨어와 하드웨어의 상호작용을 깊이 이해하는 것을 목표로 공부하고 있습니다.

특히 CPU 및 GPU 아키텍처, 메모리 계층 구조, 멀티코어 시스템에서의 성능 특성 분석에 관심이 있으며, 실제 시스템 동작을 분석하고 실험하는 과정 자체를 즐기는 편입니다.

Academic Work & Study Topics

CPU Pipeline & Hazard Analysis

Instruction-level parallelism과 pipeline hazard의 발생 원인을 분석하고, 성능에 미치는 영향을 이해하기 위한 이론 및 실험 기반 학습을 수행하였습니다.

Operating System Scheduling Study

다양한 scheduling policy의 특성을 비교하며 프로세스 실행 흐름과 context switching overhead에 대해 분석하였습니다.

Memory Hierarchy Performance Study

Cache locality, memory access pattern, latency hiding 기법 등에 대한 이해를 중심으로 시스템 성능 분석을 수행하였습니다.

Synchronization & Concurrency Concepts

Mutual exclusion, semaphore, producer-consumer problem 등 병행성 제어 기법을 학습하고 구현 실습을 진행하였습니다.
